## 1. 프로젝트 개요

**한 줄 요약:** 쿠팡 상품의 가격 변동을 추적하고, 최적의 구매 타이밍을 알려주는 크로스플랫폼 가격 알림 앱.

**비전:** 기존 경쟁 앱들의 장점을 통합하되, **"패시브 인텔리전스"** 를 핵심 차별점으로 삼는다. 사용자가 일일이 상품을 검색/즐겨찾기하지 않아도, 카테고리와 조건만 등록하면 AI가 자동으로 구매 타이밍을 판단하여 알려주는 "Set it and forget it" 가격 인텔리전스 앱.

| **가성비 직장인 준호 (28세)** | 전자기기/가전을 구매 전 가격 추이를 꼼꼼히 비교. 지금이 살 때인지 판단 근거가 필요함. | "지금이 사야 할 타이밍인지, 더 기다려야 하는지 알고 싶어" |
| **다중몰 쇼퍼 수진 (32세)** | 쿠팡뿐 아니라 11번가, G마켓도 비교하며 최저가를 찾음. 여러 앱을 오가는 것이 번거로움. | "한 앱에서 여러 쇼핑몰 가격을 한번에 비교하고 싶어" |

## 3. 해결할 문제 (기존 앱의 한계)

| 문제 | 근거 (경쟁사 분석) |
|---|---|
| **기능이 분산되어 있음** | 가격하락확률은 역대가에만, N일만에 최저가는 지니알림에만, 필터/정렬은 로우차트에만 존재. 사용자가 여러 앱을 설치해야 원하는 정보를 모두 얻을 수 있음. |
| **웹 기능 제한** | 시장 1위 폴센트는 웹에서 검색 제한, 그래프 CTA 오버레이, QR코드 유도 등으로 웹 기능을 의도적으로 제한. 앱 설치 없이 빠르게 가격을 확인하고 싶은 사용자에게 불편. |
| **쿠팡 전용** | 6개 경쟁사 중 5개가 쿠팡 전용. 유일하게 다중 쇼핑몰을 지원하는 꿀단지는 iOS 전용 + 초기 단계. |
| **구매 판단 근거 부족** | 대부분의 앱이 단순 가격 그래프만 제공. "지금이 살 때인가?"에 대한 명확한 답(확률, 요일별 패턴, 예측)을 주는 앱이 부족. |
| **사용자가 직접 다 해야 함** | 모든 경쟁 앱이 사용자가 개별 상품을 검색 → 즐겨찾기 → 알림 설정하는 수동 방식. 카테고리 단위 자동 모니터링이 없어 사용자 노력 대비 효용이 낮음. Honey의 Smart Droplist, 당근의 키워드 알림 같은 "패시브" 방식은 쇼핑 가격 추적 앱에 아직 적용되지 않음. |

## 4. 핵심 기능 (MVP)

### P0 - 필수 (출시 필수)
| 기능 | 설명 | 참고 경쟁사 |
|---|---|---|
| **가격 추적 그래프** | 상품별 시계열 가격 변동 차트. 최저가/최고가/평균가 표시. 장기 데이터(1년+) 지원. | 전 경쟁사 공통 |
| **최저가 알림** | 목표 가격 도달 시 푸시 알림. 프리셋 제공: 지정가/평균가 이하/최저가 근접/역대최저가. | 역대가(4단계 프리셋) |
| **상품 검색** | URL 입력 + 키워드 검색 모두 지원. 카테고리(16개+) 분류. | 로우차트(URL+키워드) |

### P1 - 중요 (출시 후 1개월 내)
| 기능 | 설명 | 참고 경쟁사 |
|---|---|---|
| **카테고리 패시브 알림** | 사용자가 관심 카테고리(예: 생활용품, 전자기기)를 등록하면, 해당 카테고리 내 상품의 가격 변동/구매 타이밍을 자동 모니터링하여 알림. 개별 상품 즐겨찾기 불필요. | Keepa(카테고리 Best Deal), 당근(키워드 알림) |
| **키워드 자동 모니터링** | "무선 이어폰", "기저귀" 등 키워드 등록 시 해당 키워드 상품의 가격 하락/핫딜 자동 알림. 당근마켓 키워드 알림처럼 한번 설정하면 영구 모니터링. | 당근/번개장터(키워드 알림), 폴센트(키워드 핫딜) |
| **% 기반 자동 알림** | 개별 가격 설정 없이 "5% 이상 하락 시 알림" 같은 글로벌 설정. 즐겨찾기 + 카테고리 알림 모두에 적용 가능. | Robinhood(가격 변동 %), 에누리(1% 이상 하락) |
| **가격하락확률 게이지** | 5단계 시각화 (지금사야해 ~ 절대사지마). 현재 가격 위치를 %로 표시. | 역대가(독자 기능) |
| **요일별 가격 차트** | 월~일 요일별 최저/최고 범위 + 평균 추이. 어느 요일에 사는 게 유리한지 분석. | 역대가(독자 기능) |
| **필터/정렬** | 필터: 품절임박, 역대최저, 하락추세, 만원이하. 정렬: 랭킹순, 할인률순, 할인금액순, 낮은금액순. | 로우차트 |
| **단가 계산** | 1정당, 100ml당 등 단위 가격 자동 계산 및 표시. | 로우차트, 지니알림 |

### P2 - 차별화 (출시 후 3개월 내)
| 기능 | 설명 | 참고 경쟁사 |
|---|---|---|
| **Hopper식 AI "지금 사세요/기다리세요"** | 가격 이력 + 프로모션 주기(쿠팡데이, 블프 등) + 계절성 + 판매 속도를 분석하여 "지금 구매 추천" / "N일 후 더 떨어질 확률 X%" 판단. Hopper가 항공권에서 95% 정확도로 증명한 모델. | Hopper(항공권 AI 예측) |
| **판매 속도 급증 선제 알림** | 판매량이 평소 대비 급증한 상품 자동 감지 → 품절/가격 인상 전 선제 알림. | Amazon TrendSpotter, 카카오 "급상승" 탭 |
| **다중 쇼핑몰 비교 (쿠팡 + 네이버)** | 네이버쇼핑 API(lprice) 활용 + 쿠팡파트너스 API 교차 비교. 멤버십/포인트 적립 차이까지 고려한 "실질 최저가" 계산. | 꿀단지(초기), 네이버쇼핑 API |
| **N일만에 최저가 배지** | 며칠 만의 최저가인지 구체적 숫자로 표시. 직관적 구매 긴급성 전달. | 지니알림 |
| **N원 싸게 구매하기 CTA** | 평균가 대비 절약 금액을 구매 버튼에 직접 표시. | 지니알림 |
| **카카오톡 알림 채널** | 앱 푸시 외 카카오톡 알림톡으로 가격 알림 전송. 한국 사용자 가장 친숙한 채널. | 에누리(카카오톡 알림) |

### P3 - 장기 (6개월+)
| 기능 | 설명 | 참고 서비스 |
|---|---|---|
| **AI 행동 학습 자동 추천** | 사용자 브라우징/클릭 패턴 학습 → 관심 상품 특성 파악 → 유사 딜 자동 추천. | Honey Smart Droplist, 직방 AI |

## 8. 플랫폼별 접근 전략

### 가격 추적 난이도 비교

| 플랫폼 | 가격 투명성 | 추적 난이도 | 제휴 프로그램 | 수수료 | 접근 전략 |
|--------|-----------|-----------|------------|--------|----------|
| **쿠팡** | 모든 사용자 동일 가격 | **쉬움** | 쿠팡파트너스 | 3~10% | **1순위**. 표시가격 = 실제가격. 다이내믹 프라이싱만 주기적 크롤링으로 대응 |
| **네이버쇼핑** | 표시가격 동일 (적립률만 차이) | **중간** | 쇼핑 커넥트 (2025.07~) | 5~20% + 1.8% | **2순위**. 공식 검색 API 무료 제공 (lprice/hprice). 멤버십 적립(최대 8%)은 "실질가" 계산으로 반영 |

### 쿠팡이 가격 추적에 유리한 핵심 이유
1. **단일 가격**: 로켓배송 상품은 쿠팡이 직접 가격 설정 → 로그인/회원 등급 무관 동일 가격
2. **쿠폰 영향 미미**: 웰컴/웰컴백 쿠폰은 범용적이고 표시가격에 영향 안 줌
3. **와우회원**: 상품 가격 자체는 변동 없음 (배송비/반품비 면제만)
4. **파트너스 API**: 상품 검색 + 딥링크 생성 공식 지원

### 다중 쇼핑몰 "실질 최저가" 계산 전략
- **쿠팡**: `표시가격` (가장 정확)
- **네이버**: `표시가격 - 네이버페이 적립(최대 8%)` = 실질가. 멤버십 여부로 2가지 가격 표시
- **11번가/G마켓/옥션**: `표시가격` 기준만 제공 + "쿠폰 추가 할인 가능" 표시 (정확한 최종가 보장 불가 고지)

## 9. 경쟁 우위 (vs. 폴센트)

| 항목 | 폴센트 (현 1위) | PullCents (우리) |
|---|---|---|
| **기능 범위** | 가격 그래프 + 알림 + 카드할인 | 경쟁사 장점 통합 + 패시브 알림 + AI 예측 |
| **알림 방식** | 사용자가 개별 상품 수동 등록 | **카테고리/키워드 등록 → 자동 모니터링 (패시브)** |
| **웹 접근성** | 웹 기능 심하게 제한 | 웹에서도 풀기능 (알림 설정 포함) |
| **구매 판단** | "구매타이밍" 배지 (단순) | Hopper식 "지금 사세요/기다리세요" AI 판단 |
| **쇼핑몰 범위** | 쿠팡 전용 | 쿠팡 + 네이버쇼핑 (API 기반) |
| **수익 다변화** | 쿠팡파트너스만 | 쿠팡파트너스 + 네이버 쇼핑 커넥트 |
| **사용자 노력** | 상품마다 검색 → 즐겨찾기 → 알림 설정 | 카테고리 1번 등록 → 끝 (Set it and forget it) |

**핵심 포지셔닝:** 폴센트가 "능동적 가격 알림"이라면, PullCents는 **"패시브 가격 인텔리전스"** — 사용자가 대충 지정만 하면, AI가 알아서 모니터링하고 최적 구매 타이밍을 판단하여 알려주는 앱. 사용자의 노력을 최소화하면서 절약을 극대화.

## 부록: 패시브 알림 벤치마크

"사용자가 직접 움직이지 않아도 되는" 패턴을 구현한 서비스들:

| 서비스 | 분야 | 패시브 알림 방식 | 설정 난이도 | PullCents 차용점 |
|--------|------|----------------|-----------|-----------------|
| **Hopper** | 항공권 | AI가 "지금 사세요/기다리세요" 판단 (95% 정확도) | 낮음 | **핵심 모델**. 가격 이력+계절성+프로모션 주기 분석 |
| **Honey Smart Droplist** | 쇼핑 | 자주 보는 상품 자동 추적 (설정 필요 없음) | 제로 | 브라우징 행동 자동 감지 (P3) |
| **당근 키워드 알림** | 중고거래 | 키워드 1개 입력 → 영구 모니터링 | 극히 낮음 | 키워드 기반 카테고리 알림 (P1) |
| **Robinhood** | 주식 | 5%/10% 변동 시 자동 알림 + 52주 최저 알림 | 낮음 | % 기반 글로벌 알림 설정 (P1) |
| **직방 AI** | 부동산 | 클릭 패턴 학습 → 유사 매물 자동 추천 | 낮음 | AI 행동 학습 자동 추천 (P3) |
| **에누리** | 가격비교 | 24시간 내 1% 이상 하락 시 카카오톡 알림 | 낮음 | 카카오톡 알림 채널 (P2) |
| **Amazon Rufus** | 쇼핑 | "30% 할인되면 자동 구매" AI 에이전트 | 낮음 | 조건부 원클릭 구매 링크 (P3) |
