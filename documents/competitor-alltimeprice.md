## 상품 상세 페이지

### 상품 정보 영역
- 카테고리 경로 (빵부스러기: 식품 > 수산물/건어물 > 생선 > ...)
- 별점 + 리뷰수
- 상품명
- 옵션 드롭다운
- **현재가격** (굵은 글씨) + 할인율/할인금액 (빨간색)
- 카카오톡 공유 버튼 (TALK 아이콘)

### 가격 분석 영역
- **역대최고가**: 빨간색 금액
- **최근평균가**: 일반 텍스트
- **역대최저가**: 빨간색 금액

### ★ 가격하락확률 게이지바 (독특한 기능)
- 5단계 게이지: 지금사야해 ← 구입해봐요 ← 생각해봐요 ← 사지마세요 ← **절대사지마**
- 현재 위치를 퍼센트(%)로 표시 (예: 94%)
- 시각적으로 구매 타이밍 판단 도움

### 알림 설정 (4단계)
1. **지정가이하**: 사용자 입력 금액 → "알림받기"
2. **평균가이하**: 자동 계산 → "알림받기"
3. **최저가근접**: 자동 계산 → "알림받기"
4. **역대최저가**: 자동 계산 → "알림받기"

### 쿠팡파트너스 고지
- **"이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다."**
- "쿠팡사러가기" CTA 버튼 (빨간색, 전체 너비)

### 가격추적 그래프
- **Highcharts** 라이브러리 사용
- 시계열: 2022년 5월 ~ 현재 (장기 추적)
- 범례: 현재가(빨간 점) / 판매중(검정선) / 품절(빨간선)
- 역대최고가/역대최저가 라벨 표시
- 점선: 평균가 기준선

### ★ 요일별 가격 차트 (독특한 기능)
- 월~일 요일별 최저가/최고가 범위 (막대)
- 평균 추이선 (녹색 라인)
- **언제 사는 게 가장 싼지** 한눈에 파악 가능

### 하단
- "함께 비교하면 좋은 상품" 추천 슬라이더

## 특징 / 차별점
1. **가격하락확률 게이지** (5단계 시각화) — 다른 앱에 없는 독자 기능
2. **요일별 가격 차트** — 어느 요일에 사는 게 유리한지 분석
3. **4단계 알림 프리셋** (지정가/평균가/최저가근접/역대최저가)
4. **장기 가격 데이터** (2022년부터 추적)
5. **미니 추이 아이콘** (상품 카드에 시각적 트렌드)
6. **쿠팡 프로모션 배너** 직접 노출 (와우회원 등)
7. 카카오톡 공유 기능

## 수익모델
