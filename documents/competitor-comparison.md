## 8. SWOT 종합 (PullCents가 노려야 할 포인트)

### 기회 (Opportunity)
- **폴센트 약점**: 웹 기능 제한 심함 → 웹에서도 풀기능 제공하면 차별화
- **역대가 강점 차용**: 가격하락확률 게이지 + 요일별 가격 차트 → 구매 의사결정 도움
- **지니알림 강점 차용**: "N일만에 최저가" + "N원 싸게 구매" → 직관적 절약 금액 표시
- **로우차트 강점 차용**: 필터/정렬 + 단가 계산 + 3일간 하락률
- **유료 구독**: 폴센트만 유일하게 시도 중 → 프리미엄 기능 확대 가능

### 위협 (Threat)
- 폴센트의 350만 DL + 4.9 평점 (압도적 1위)
- 쿠팡파트너스 정책 변경 리스크
- 유사 앱 포화 (5개+ 이미 존재)

### 차별화 전략 제안
1. **모든 경쟁사 장점 통합** — 가격 그래프 + 하락확률 + 요일별 + 단가 + 필터
2. **웹 풀기능** — 앱 설치 없이도 알림 설정 가능 (로우차트처럼)
3. **AI 가격 예측** — 향후 가격 하락 가능성 예측 (독자 기능)
4. **다중 쇼핑몰 비교** — 쿠팡 외 네이버쇼핑/11번가 등 크로스 비교
5. **소셜 기능** — 가격 정보 공유, 위시리스트 공유 커뮤니티
