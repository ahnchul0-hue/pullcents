## 2. 화면 흐름도 (Screen Flow)

### 2.1 전체 네비게이션 맵

```
                         ┌─────────────────────────────────────────────┐
                         │              APP / WEB 진입                 │
                         └─────────────┬───────────────────────────────┘
                                       │
                    ┌──────────────────┼──────────────────┐
                    ▼                  ▼                  ▼
             [ONBOARDING]        [WEB_LANDING]      [AUTH_LOGIN]
             (최초 1회)          (비로그인 웹)       (로그인 필요 시)
                    │                  │                  │
                    └──────────────────┼──────────────────┘
                                       ▼
    ┌──────────────────────────────────────────────────────────────────┐
    │                    ══ 하단 탭바 (앱) / 상단 GNB (웹) ══          │
    │                                                                  │
    │  [HOME]      [SEARCH]     [FAVORITES]   [ALERT_CENTER]  [MY]    │
    │   홈           검색         즐겨찾기        알림센터      마이    │
    └──┬──────────────┬──────────────┬──────────────┬──────────┬───────┘
       │              │              │              │          │
       ▼              ▼              │              ▼          ▼
  [CATEGORY]    [SEARCH_RESULT]     │         (알림 탭)   [MY_PAGE]
       │              │              │              │          │
       └──────┬───────┘              │              │     ┌────┼────┐
              ▼                      │              │     ▼    ▼    ▼
       [PRODUCT_DETAIL] ◄───────────┘──────────────┘  [SUB] [SET] [KEYWORD]
              │
       ┌──────┼──────┐
       ▼      ▼      ▼
  [ALERT]  [SHARE]  [쿠팡 외부]
  알림설정  공유    구매하기
```

### 2.2 시나리오 1: 상품 검색 → 가격 확인 → 알림 설정

```
사용자 의도: "에어팟 가격이 떨어지면 알려줘"

  [HOME]
    │
    │  상단 검색바 탭
    ▼
  [SEARCH]
    │  "에어팟" 입력  ← 인기검색어 참고 가능
    │  (또는 쿠팡 URL 붙여넣기)
    ▼
  [SEARCH_RESULT]
    │  필터: 역대최저 / 하락추세
    │  정렬: 할인률순
    │  상품 카드 탭  ← "23일만에 최저가" 배지 확인
    ▼
  [PRODUCT_DETAIL]
    │  ┌─ 가격 그래프 확인 (시계열 + 요일별)
    │  ├─ 가격하락확률 게이지 확인 (예: 94% "지금사야해")
    │  ├─ 최저가↔최고가 슬라이더로 현재 위치 파악
    │  └─ "알림 설정" 버튼 탭
    ▼
  [ALERT_SETTING]  (바텀시트 또는 인라인)
    │  4단계 프리셋 중 선택:
    │    ○ 지정가 이하 (직접 입력)
    │    ○ 평균가 이하
    │    ○ 최저가 근접
    │    ○ 역대 최저가
    │  "알림 받기" 버튼 탭
    ▼
  [완료]  ← 즐겨찾기에 자동 추가 + 확인 토스트
```

### 2.3 시나리오 2: 알림 수신 → 상품 확인 → 쿠팡 구매

```
사용자 의도: "알림 왔네, 싸졌으면 바로 살래"

  [푸시 알림 / 웹 알림]
    │  "에어팟 Pro 2 — 역대최저가 도달! 219,000원 (-15%)"
    │  알림 탭
    ▼
  [PRODUCT_DETAIL]
    │  ┌─ 현재가 확인: 219,000원
    │  ├─ 가격하락확률 게이지: "지금사야해" (97%)
    │  ├─ "35,000원 싸게 구매하기" CTA 확인
    │  └─ 하단 고정 CTA "쿠팡에서 구매하기" 탭
    ▼
  [쿠팡 외부 이동]  ← 쿠팡파트너스 링크 경유
    │  구매 완료
    ▼
  [커미션 발생]  ← 24시간 내 구매 시 ~3% 수익
```

### 2.4 시나리오 3: 홈 브라우징 → 핫딜 발견 → 구매

```
사용자 의도: "뭐 싸게 파는 거 없나 구경하자"

  [HOME]
    │  ┌─ 필터 탭: "역대최저" 선택
    │  ├─ 카테고리: "가전/디지털" 선택
    │  └─ 정렬: "할인률순" 선택
    ▼
  [HOME - 필터 적용됨]
    │  상품 카드 스크롤 브라우징
    │  ← 미니 추이 아이콘으로 가격 트렌드 즉시 파악
    │  ← "N일만에 최저가" 배지 눈에 띔
    │  상품 카드 탭
    ▼
  [PRODUCT_DETAIL]
    │  ┌─ 요일별 차트로 "오늘이 제일 싼 요일" 확인
    │  ├─ 다른 옵션 비교 → 더 저렴한 옵션 발견
    │  └─ "N원 싸게 구매하기" CTA 탭
    ▼
  [쿠팡 외부 이동]
```

### 3.2 상품 카드 (Product Card Component)

**설계 근거**: 지니알림의 "N일만에 최저가" + 역대가의 미니 추이 아이콘 + 로우차트의 단가/하락률 통합

```
┌──────────────────────────────┐
│ ┌──────────────────────────┐ │
│ │                          │ │
│ │      상품 이미지          │ │
│ │                          │ │
│ │ ┌──────────────┐         │ │  ← 좌상단 배지
│ │ │ 23일만에 최저가│         │ │     (지니알림 차용)
│ │ └──────────────┘         │ │
│ └──────────────────────────┘ │
│                              │
│  상품명 최대 2줄까지 표시... │
│                              │
│  ▼15%  19,900원  ~~23,400~~  │  ← 할인율 + 현재가 + 원가
│  (1정당 199원)               │  ← 단가 (로우차트/지니알림 차용)
│                              │
│  ┌────┐  3,500원 하락        │  ← 미니 추이 아이콘 (역대가 차용)
│  │ 📉 │  3일간 -8.2%         │     + 하락금액 + 3일 하락률
│  └────┘                      │     (로우차트 차용)
│                              │
│  ★ 4.8 (3,241)    🚀로켓     │  ← 평점/리뷰 + 로켓배송
│                              │
│  [역대최저] [하락추세]        │  ← 태그 (로우차트 차용)
└──────────────────────────────┘
```

**배지 우선순위 규칙:**
1. "역대최저가" (빨간색) — 역대 최저가 도달 시
2. "N일만에 최저가" (파란색) — N >= 7일일 때 표시
3. "최저가근접" (노란색) — 역대 최저가 대비 5% 이내
4. "하락추세" (초록색) — 3일 연속 하락 시
5. "품절임박" (주황색) — 재고 적을 때

## 4. 네비게이션 구조

### 4.1 하단 탭바 (앱)

```
┌────────────────────────────────────────────────────────────┐
│  🏠 홈    🔍 검색    ⭐ 즐겨찾기    🔔 알림    👤 마이    │
│  Home    Search   Favorites   Alerts    My      │
└────────────────────────────────────────────────────────────┘
```

| 탭 | 아이콘 | 이동 화면 | 배지 |
|---|---|---|---|
| 홈 | 집 | `HOME` | 없음 |
| 검색 | 돋보기 | `SEARCH` | 없음 |
| 즐겨찾기 | 별 | `FAVORITES` | 즐겨찾기 수 |
| 알림 | 종 | `ALERT_CENTER` | 미읽은 알림 수 (빨간 점) |
| 마이 | 사람 | `MY_PAGE` | 없음 |

**탭 5개 선정 근거:**
- 홈/검색/마이 : 필수 3탭 (모든 경쟁사 공통)
- 즐겨찾기 : 폴센트/역대가에서 핵심 기능으로 확인 (관심 상품 빠른 접근)
- 알림 : 가격 알림이 앱의 핵심 가치이므로 탭 레벨로 승격 (지니알림의 벨 아이콘 참고)

### 4.2 상단 헤더 구성

**앱 헤더 (홈)**
```
┌────────────────────────────────────────────────────────────┐
│  PullCents                             [🔔 3]  [👤]       │
├────────────────────────────────────────────────────────────┤
│  🔍 검색바 (탭 시 검색 화면으로 이동)                      │
└────────────────────────────────────────────────────────────┘
```

**앱 헤더 (상세 페이지)**
```
┌────────────────────────────────────────────────────────────┐
│  ← 뒤로      [페이지 타이틀]       [공유] [♡]             │
└────────────────────────────────────────────────────────────┘
```

**웹 헤더 (GNB)**
```
┌─────────────────────────────────────────────────────────────────────┐
│  [로고] PullCents   🔍 [검색바 (넓게)]   [로그인] [회원가입]       │
│                                                                     │
│  홈 | 카테고리 ▼ | 핫딜 | 키워드알림 | 즐겨찾기 | 알림센터 | 마이   │
└─────────────────────────────────────────────────────────────────────┘
```

### 4.3 웹 vs 앱 UI 차이점

| 요소 | 앱 | 웹 |
|---|---|---|
| **네비게이션** | 하단 탭바 (5탭) | 상단 GNB (수평 메뉴) |
| **카테고리** | 가로 스크롤 태그 바 | 좌측 고정 사이드바 (로우차트 방식) |
| **상품 그리드** | 2열 | 3~4열 (반응형) |
| **검색** | 전체화면 전환 | 헤더 내 인라인 |
| **알림 설정** | 바텀시트 모달 | 인라인 폼 (페이지 내) |
| **가격 그래프** | 세로 스크롤 (풀폭) | 가로 넓게 (더 많은 데이터 포인트) |
| **하단 CTA** | 고정 하단바 | 우측 사이드 고정 또는 스크롤 따라감 |
| **기능 제한** | 없음 (풀기능) | **없음 (풀기능)** ← 폴센트와 핵심 차별점 |
| **알림 방식** | 푸시 알림 | 웹 푸시 + 이메일 (선택) |
| **앱 유도** | 해당없음 | 부드러운 유도 (팝업 아님, 배너만) |

**핵심 차별점: 웹에서도 풀기능 제공**
> 근거: 폴센트는 웹에서 검색 제한, 그래프 가림, QR코드 고정 등 공격적 앱 유도 전략을 사용함.
> 로우차트는 웹에서 알림 설정까지 가능하여 사용자 경험이 우수함.
> PullCents는 로우차트 방식을 따르되, 웹 푸시 알림까지 지원하여 앱 설치 없이도 핵심 가치를 제공.

## 5. 경쟁사 대비 UI 차별점

### 5.1 경쟁사별 차용 요소 정리

| 차용 요소 | 출처 | PullCents 적용 위치 | 근거 |
|---|---|---|---|
| 가격하락확률 게이지 (5단계) | **역대가** | 상품 상세 | 구매 의사결정에 직관적 도움. 경쟁사 중 유일한 독자 기능 |
| 요일별 가격 차트 | **역대가** | 상품 상세 | "언제 사야 싼지" 답변 제공. 다른 앱에 없음 |
| 4단계 알림 프리셋 | **역대가** | 알림 설정 | 사용자가 원하는 수준 선택 가능. 단순 지정가보다 편리 |
| 미니 추이 아이콘 | **역대가** | 상품 카드 | 리스트에서 가격 트렌드 즉시 파악 |
| "N일만에 최저가" 배지 | **지니알림** | 상품 카드/상세 | 구체적 숫자로 희소성 전달. "최저가" 단어보다 강력 |
| 최저가↔최고가 슬라이더 | **지니알림** | 상품 상세 | 현재가 위치를 직관적으로 시각화 |
| "N원 싸게 구매하기" CTA | **지니알림** | 하단 고정 CTA | 절약 금액 명시로 구매 전환률 향상 |
| 인기검색어 태그 | **지니알림** | 검색 화면/홈 | 검색 UX 향상 + 트렌드 노출 |
| 4종 필터 (품절임박/역대최저/하락추세/만원이하) | **로우차트** | 홈/검색결과 | 상품 발견 효율 극대화 |
| 4종 정렬 (랭킹/할인률/할인금액/낮은금액) | **로우차트** | 홈/검색결과 | 다양한 관점에서 상품 비교 |
| 단가 계산 (1정당, 100ml당) | **로우차트/지니알림** | 상품 카드/상세 | 대용량 상품 비교 시 필수 |
| 3일간 하락률 | **로우차트** | 상품 카드 | 최근 가격 변동 추세 즉시 파악 |
| 좌측 카테고리 사이드바 | **로우차트** | 웹 카테고리 | 웹에서 편리한 카테고리 탐색 |
| 웹 알림 설정 | **로우차트** | 상품 상세 (웹) | 앱 설치 없이 핵심 기능 사용 |
| 카드 할인가 적용 | **폴센트** | 상품 상세 (v2) | 실제 결제 금액 파악. 1차 출시 후 추가 검토 |
| 키워드 핫딜 추적 | **폴센트** | 키워드 알림 화면 | 특정 키워드 관련 핫딜 자동 추적 |
| 다른 옵션 비교 | **폴센트/지니알림** | 상품 상세 | 같은 상품의 다른 옵션 가격 한눈에 비교 |

### 5.2 PullCents 고유 UI 요소 (차별화)

경쟁사에 없는 PullCents만의 고유 기능/UI 제안:

#### 1) AI 가격 예측 위젯
```
┌─── AI 가격 예측 (PREMIUM) ─────────────────────┐
│                                                  │
│  📈 향후 7일 예측                                │
│                                                  │
│  현재  219,000원                                 │
│  3일후 215,000원 예측 (▼1.8%)   신뢰도: 72%     │
│  7일후 222,000원 예측 (▲1.4%)   신뢰도: 58%     │
│                                                  │
│  💡 3일 후 추가 하락 가능성이 있습니다.          │
│     조금 더 기다려보는 것을 추천합니다.           │
│                                                  │
└──────────────────────────────────────────────────┘
```
> 근거: 역대가의 가격하락확률 게이지가 과거 데이터 기반이라면, PullCents는 ML 모델로 미래 예측까지 제공. 경쟁사 중 시도한 곳 없음. 프리미엄 전용으로 구독 전환 유도.

#### 2) 가격 변동 타임라인 (소셜 피드 형태)
```
┌─── 가격 변동 타임라인 ──────────────────────────┐

│  │                                               │

│  │                                               │
│  ▼ 더보기                                        │
└──────────────────────────────────────────────────┘
```
> 근거: 그래프만으로는 품절/재입고 같은 이벤트를 파악하기 어려움. 타임라인 형태로 주요 이벤트를 시간순 나열하면 이해도 향상.

> 근거: 로우차트만 웹 알림을 지원하지만 이메일 알림은 없음. 웹 푸시 + 이메일 이중 채널로 알림 도달률 극대화. 앱 미설치 사용자도 핵심 가치 경험 가능.
