## 웹사이트 구조

### 헤더
- 로고 (로우차트)
- 검색바: "쿠팡 상품URL 또는 검색키워드를 입력하세요."
- "처음오셨나요?" 링크
- 로그인 / 회원가입

### 메인 페이지
1. **상단 인기 상품 슬라이더** (4개 상품, 자동 슬라이드)
2. **필터 라디오 버튼**: 품절임박 / 역대최저 / 하락추세 / 만원이하
3. **정렬 옵션**: 랭킹순, 할인률순, 할인금액순, 낮은금액순
4. **좌측 카테고리 사이드바** (항상 표시)
5. **상품 카드 그리드** (4열)

### 좌측 카테고리 (18개+)
전체 카테고리, 생활용품, 반려동물용품, 뷰티, 문구/오피스, 가전디지털, 자동차용품, 홈인테리어, 주방용품, 식품, 출산/유아동, 헬스/건강식품(비타민/미네랄, 건강식품, 허브/식물추출물, 홍삼/인삼, 건강즙/음료, 꿀/프로폴리스, 건강분말/건강환, 헬스보충식품, 다이어트/이너뷰티, 영양식/선식, 어린이 건강식품, 홈트레이닝, 헬스/요가용품, 건강가전, 건강도서, 건강/의료용품), 도서/음반/DVD, 스포츠/레저, 완구/취미

### 상품 카드 UI
- 할인율 배지 (좌상단, 예: "20%")
- 상품 이미지
- 상품명 (2줄)
- 할인율 + 원래가격(취소선) → **현재가격** (빨간색)
- (1정당 금액) 단가 정보
- 하락 금액 정보 (예: "16,500원 하락")
- 태그: 역대최저가 / 하락추세
- "3일간 하락률 25%" 정보
- 별점 + 리뷰수

## 상품 상세 페이지

### 상품 정보 영역
- 상품 이미지 (좌)
- 상품명
- 별점 + 리뷰수
- 할인율 + 원래가격(취소선) → **현재가격** (1정당 금액)
- 하락 금액
- 태그: 최대적립 금액, 역대최저, 하락추세, "3일간 하락률 25%"
- **"쿠팡에서 보기"** 버튼 (파란색) + "공유하기" 버튼
- 옵션 드롭다운 (상품 옵션 선택)

### 가격 알림 설정
- "가격이 [알림 받을 가격] 이하로 내려가면" → **알림** 버튼
- 웹에서 직접 알림 설정 가능 (폴센트와 차별점)

### 가격 그래프
- 시계열 라인 차트
- 하락 금액 표시 (예: "30원 하락", "3,870원 하락", "12,600원 하락")
- 평균가 점선 + "현재/역대최저가" 라벨 (우측)
- 현재가격 빨간 점선
- 워터마크: "로우차트" (그래프 중앙)
- X축: 날짜 (일 단위)

### 하단
- "쿠팡에서 상품정보 더보기" 링크
- **관련 상품 추천** (하단 이미지 슬라이더)
- **쿠팡 광고 배너** ("주문하기 >" 버튼)

## 특징 / 차별점
1. **웹에서 가격 알림 설정 가능** (앱 설치 불필요)
2. **필터 기능**: 품절임박/역대최저/하락추세/만원이하
3. **정렬 기능**: 랭킹순/할인률순/할인금액순/낮은금액순
4. **3일간 하락률** 표시
5. **단가 계산** (1정당, 100ml당 등)
6. **좌측 고정 카테고리 사이드바** + 하위 카테고리 트리
7. **회원 시스템** (로그인/회원가입)
8. **북마클릿** 기능 (타 앱에서 열기)
9. 쿠팡 광고 배너 직접 노출

## 수익모델
- 쿠팡파트너스 링크 (쿠팡에서 보기/구매 버튼)
- 쿠팡 광고 배너
