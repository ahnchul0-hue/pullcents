## 상품 상세 페이지

### 상품 정보 영역
- 할인율 배지 (빨간색, 예: "▼12%")
- **"역대최저가😎"** 배지
- 상품명: [쿠팡] + 상품명
- 옵션 정보 (옵션 _ 개당 캡슐/정 ★ 수량 ★ 개수)
- (1정당 금액) 단가 정보
- 현재가: 원래가(취소선) → **현재가격**
- **공유하기** 버튼

### 가격 분석
- **평균구매가**: 금액
- **역대최고가**: 빨간색 금액
- **역대최저가**: 빨간색 금액

### ★ 최저가↔최고가 슬라이더 게이지 (독특한 기능)
- 좌측 "최저가" — 우측 "최고가"
- 현재가 위치를 슬라이더로 시각화
- "현재가" 라벨 표시

### 가격 그래프
- Area 차트 (영역 채우기, 하늘색)
- 날짜 기반 (월-일 형식)
- 호버 시 날짜 + 가격 툴팁 표시
- 약 6개월치 데이터

### 하단 영역
- **해당 상품의 다른옵션 🔗**: 다른 옵션 리스트
- "앱에서 원하는 가격에 알림을 설정하시면 더 편리하게 이용하실 수 있어요! 🎁" 앱 유도 배너

### 하단 고정 CTA 바
- **"🔔 알람설정"** 버튼 (파란색)
- **"🛒 828원 싸게 구매하기"** 버튼 (회색) — 평균가 대비 절약 금액 표시

## 특징 / 차별점
1. **"N일만에 최저가"** 배지 — 며칠 만의 최저가인지 구체적 표시
2. **최저가↔최고가 슬라이더** — 현재가 위치 직관적 시각화
3. **"N원 싸게 구매하기"** CTA — 절약 금액을 구매 버튼에 직접 표시
4. **인기 검색어 태그** — 트렌드 검색어 표시
5. **Area 차트** — 영역 채우기 스타일 (다른 앱과 시각 차별)
6. **이모지 활용** — 배지/텍스트에 이모지 적극 사용 (친근함)
7. 웹에서 알림 설정 가능
