## 1. 프로젝트 개요

**한 줄 요약:** 쿠팡 상품의 가격 변동을 추적하고, 최적의 구매 타이밍을 알려주는 **완전 무료** 크로스플랫폼 가격 알림 앱.

**비전:** 기존 경쟁 앱들의 장점을 통합하되, **"패시브 인텔리전스"** 를 핵심 차별점으로 삼는다. 사용자가 일일이 상품을 검색/즐겨찾기하지 않아도, 카테고리와 조건만 등록하면 AI가 자동으로 구매 타이밍을 판단하여 알려주는 "Set it and forget it" 가격 인텔리전스 앱.

**수익 구조:** 사용자가 앱을 통해 구매하면 쿠팡파트너스(3~10%) + 네이버 쇼핑 커넥트(5~20%) 커미션이 발생하므로, 모든 기능을 무료로 제공하면서도 수익을 확보할 수 있다. 유료 구독 없이 사용자 경험에 집중한다.

## 3. 해결할 문제 (기존 앱의 한계)

| 문제 | 근거 (경쟁사 분석) |
|---|---|
| **기능이 분산되어 있음** | 가격하락확률은 역대가에만, N일만에 최저가는 지니알림에만, 필터/정렬은 로우차트에만 존재. 사용자가 여러 앱을 설치해야 원하는 정보를 모두 얻을 수 있음. |
| **웹 기능 제한** | 시장 1위 폴센트는 웹에서 검색 제한, 그래프 CTA 오버레이, QR코드 유도 등으로 웹 기능을 의도적으로 제한. 앱 설치 없이 빠르게 가격을 확인하고 싶은 사용자에게 불편. |
| **쿠팡 전용** | 6개 경쟁사 중 5개가 쿠팡 전용. 유일하게 다중 쇼핑몰을 지원하는 꿀단지는 iOS 전용 + 초기 단계. |
| **구매 판단 근거 부족** | 대부분의 앱이 단순 가격 그래프만 제공. "지금이 살 때인가?"에 대한 명확한 답(확률, 요일별 패턴, 예측)을 주는 앱이 부족. |
| **사용자가 직접 다 해야 함** | 모든 경쟁 앱이 사용자가 개별 상품을 검색 → 즐겨찾기 → 알림 설정하는 수동 방식. 카테고리 단위 자동 모니터링이 없어 사용자 노력 대비 효용이 낮음. |
| **유료 벽 뒤의 기능** | 폴센트는 프리미엄 구독(연 ~9,900원)으로 AI 예측, 알림 개수 확대 등 핵심 기능을 제한. 사용자가 돈을 내야만 제대로 쓸 수 있는 구조. |

## 5-1. 센트(¢) 보상 체계

### 보상 화폐: "센트(¢)"

PullCents의 앱 내 보상 화폐. "Pull + Cents"에서 직접 유래. 사용자가 센트를 모아 **실물 기프티콘**으로 교환하는 구조로, 기존 앱들의 "의미 없는 포인트"가 아닌 **실질적 가치**를 제공한다.

### 센트(¢) 획득 채널

| 채널 | 획득량 | 조건 | 지급 방식 |
|------|--------|------|----------|
| **친구 초대** | 3¢ / 명 | 피초대자 가입 완료 시 | **확정 지급** |
| **친구 초대 (피초대자)** | 1¢ | 추천 코드로 가입 시 | **확정 지급 (웰컴)** |
| **출석체크** | 0~1¢ / 10일 | 10일 연속 출석 달성 | **확률형 룰렛** (월 최대 3회) |
| **이벤트/퀴즈** | 0~2¢ / 참여 | 정답 또는 참여 완료 시 | **확률형 룰렛** |

### 출석체크 룰렛 규칙

- 매일 앱 방문 시 출석 기록 (연속 일수 카운트)
- **10일 연속 출석** 달성 시 룰렛 1회 기회 획득
- 룰렛 결과: 당첨(1¢) 또는 미당첨(0¢)
- **월 최대 3회** 룰렛 기회 (30일 기준, 10일×3)
- 연속 출석 끊기면 카운트 리셋

### 이벤트/퀴즈 룰렛 규칙

- 가격 맞추기 퀴즈, 설문 참여 등 이벤트 참여 완료 시 **룰렛 기회 획득**
- 직접 센트 지급 없음 — 항상 확률형 룰렛을 통한 보상
- 룰렛 결과: 0¢ / 1¢ / 2¢ (확률 분포는 운영 조절)
- 이벤트 빈도: 주 1~2회 이벤트 운영 목표

### 기프티콘 교환 (센트 사용)

| 상품 예시 | 교환 가격 | 비고 |
|-----------|----------|------|
| CU 편의점 아메리카노 | 2¢ | 최소 교환 단위 |
| 배스킨라빈스 싱글킹 | 3¢ | |
| 스타벅스 톨 아메리카노 | 3¢ | |
| 기타 기프티콘 | 2~5¢ | **동적 가격** — 실제 기프티콘 시세 연동 |

**동적 가격 전략:** 기프티콘의 실제 도매가(기프티쇼, 카카오선물하기 등)를 주기적으로 반영하여 교환 가격을 자동 조정. 운영 마진을 확보하면서도 사용자에게 실질적 혜택 제공.

### 센트 경제 설계 원칙

1. **확률형 보상으로 리텐션 극대화** — 확정 보상보다 룰렛이 심리적 기대감과 재방문 동기를 더 강하게 유발
2. **친구 초대만 확정 지급** — 바이럴 성장이 가장 중요하므로 유일하게 확정 보상
3. **기프티콘 = 실질 가치** — "쓸 데 없는 포인트"가 아닌 실제 상품 교환으로 사용자 만족도 확보
4. **동적 가격 = 비용 통제** — 기프티콘 시세 변동에 따라 교환 비율 자동 조정, 운영 비용 예측 가능
5. **월 최대 지급 상한** — 출석 3¢/월 + 이벤트 α = 예측 가능한 보상 지출

### 가격 추적 난이도 비교

| 플랫폼 | 가격 투명성 | 추적 난이도 | 제휴 프로그램 | 수수료 | 접근 전략 |

| **옥션** | G마켓과 동일 구조 | **어려움** | 없음 | - | **3순위 (가격비교용)** |

| ~~티몬~~ | - | - | - | - | ~~영업 무기한 중단. 제외~~ |

### 쿠팡이 가격 추적에 유리한 핵심 이유
1. **단일 가격**: 로켓배송 상품은 쿠팡이 직접 가격 설정 → 로그인/회원 등급 무관 동일 가격
2. **쿠폰 영향 미미**: 웰컴/웰컴백 쿠폰은 범용적이고 표시가격에 영향 안 줌
3. **와우회원**: 상품 가격 자체는 변동 없음 (배송비/반품비 면제만)
4. **파트너스 API**: 상품 검색 + 딥링크 생성 공식 지원

### 다중 쇼핑몰 "실질 최저가" 계산 전략
- **쿠팡**: `표시가격` (가장 정확)
- **네이버**: `표시가격 - 네이버페이 적립(최대 8%)` = 실질가
- **11번가/G마켓/옥션**: `표시가격` 기준 + "쿠폰 추가 할인 가능" 표시

## 9. 경쟁 우위 (vs. 폴센트)

| 항목 | 폴센트 (현 1위) | PullCents (우리) |
|---|---|---|
| **가격** | 프리미엄 연 ~9,900원 (핵심 기능 유료) | **완전 무료** (모든 기능 개방) |
| **기능 범위** | 가격 그래프 + 알림 + 카드할인 | 경쟁사 장점 통합 + 패시브 알림 + AI 예측 + 센트(¢) 보상 + 기프티콘 교환 |
| **알림 방식** | 사용자가 개별 상품 수동 등록 | **카테고리/키워드 등록 → 자동 모니터링 (패시브)** |
| **알림 제한** | 무료 5개, 프리미엄 무제한 | **무제한 (무료)** |
| **웹 접근성** | 웹 기능 심하게 제한 | 웹에서도 풀기능 |
| **구매 판단** | "구매타이밍" 배지 (단순) | Hopper식 AI 판단 **(무료)** |
| **쇼핑몰 범위** | 쿠팡 전용 | 쿠팡 + 네이버쇼핑 |
| **사용자 노력** | 상품마다 검색 → 즐겨찾기 → 알림 설정 | 카테고리 1번 등록 → 끝 |

**핵심 포지셔닝:** 폴센트가 유료 구독으로 기능을 제한하는 동안, PullCents는 **모든 기능을 무료로 개방**하고 커미션 수익으로 운영한다. 사용자가 더 많이, 더 자주 구매할수록 우리도 수익이 늘어나는 **이해 일치(aligned incentives)** 구조.
